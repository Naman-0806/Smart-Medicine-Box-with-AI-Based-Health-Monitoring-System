# 💊 Smart Medicine Box

An IoT-based Smart Medicine Box that combines intelligent medicine reminders, real-time health monitoring, cloud connectivity, and an interactive dashboard to improve medication adherence and patient care.

## 📖 Overview

The Smart Medicine Box is designed to help patients take their medicines on time while continuously monitoring vital health parameters. The system integrates hardware (ESP32 and sensors) with a Python-based dashboard to provide real-time insights for patients, caregivers, and healthcare professionals.

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/<your-username>/Smart-Medicine-Box.git
```

2. Navigate to the project folder

```bash
cd Smart-Medicine-Box
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
